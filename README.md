# My personal dotfiles

## Prerequisities

Programs:
* Foot terminal
* Sway Wayland compositor
* Waybar Sway extension

Resources:
* JetBrains Mono font
* FontAwesome font

## Setup

```bash
printf "\nsource .bashps1\n" >> .bashrc
chmod +x .start
```

## How to start the desktop environment

```bash
./.start
```
